Live stream

Live streaming protocols

  • HLS: HTTP Live Streaming (old Safari/Edge)
  • DASH: Dynamic Adaptive Streaming over HTTP (all other browsers)

The idea

  • Server implements DASH protocol
  • Client listens using DASH client library
